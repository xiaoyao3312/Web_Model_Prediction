Web_Learning_Model 專案目錄結構說明

Web_Model_Prediction/
│
├─ app.py                                # Flask 主程式入口
├─ config.py                             # 應用程式配置 (目前未實作，但建議保留)
├─ requirements.txt                      # Python 依賴清單 (已生成)
├─ 檔案操作手冊.txt                      # 專案文件 (已生成)
├─ 檔案架構.txt                          # 本文件
│
├─ routes/                               # 路由 Blueprint 模組
│  └─ churn_bank_routes.py              # 銀行流失專案 API 邏輯 (包含模擬預測和 Gemini API 呼叫)
│
├─ data/                                 # 數據資產存放區
│  ├─ models/                           # 存放模型權重檔案 (.pkl, .cbm, etc.)
│  │  └─ placeholder_models.txt         # (佔位符)
│  └─ datasets/                         # 存放小型資料集
│     └─ placeholder_datasets.txt       # (佔位符)
│
├─ projects/                             # 存放模型開發/訓練原始碼 (非 Web 運行代碼)
│  ├─ Churn_Bank_code/                  # 銀行流失專案原始碼
│  │  └─ Churn_Bank_analysis.ipynb      # (佔位符)
│  └─ Churn_Telco_code/                 # 電信流失專案原始碼
│     └─ Churn_Telco_script.py          # (佔位符)
│
├─ templates/                            # HTML 模板文件 (Jinja2)
│  ├─ index.html                        # 網站主頁
│  ├─ base_Churn.html                   # 網站結構通用基礎模板
│  ├─ Churn_Bank.html                   # 銀行流失專案頁面
│  └─ Churn_Telco.html                  # 電信流失專案頁面
│
└─ static/                               # 靜態資源 (CSS, JS, 圖片)
├─ css/                              # 樣式表
│  ├─ Color_BG_Control.css           # (建議未來實作)
│  ├─ base_Churn.css                 # 頁面專用樣式 (三面板佈局)
│  ├─ Churn_Bank.css                 # 頁面專用樣式x加入自己的
│  ├─ Churn_Telco.css                # 頁面專用樣式x加入自己的
│  └─ index_html.css                 # 主頁面專用樣式
│
├─ img/                              # 圖片資源 (佔位符)
│  ├─ logo.png                       # 網站 Logo
│  ├─ Churn_Bank.jpg                 # 銀行模型卡片圖片
│  └─ Churn_Telco.jpg                # 電信模型卡片圖片
│
└─ js/                               # JavaScript 腳本
├─ Color_BG_Control.js            # (建議未來實作)
├─ base_Churn.js                  # 核心共用邏輯 (AJAX 模板, UI控制)
├─ Churn_Bank.js                  # 核心共用邏輯x銀行流失頁面專有邏輯
└─ Churn_Telco.js                 # 核心共用邏輯x電信流失頁面專有邏輯
