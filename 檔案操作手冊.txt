專案操作手冊 - Web 模型預測平台
建議使用 ANACONDA 下載 VSCODE 的 PYTHON 3.11.14 環境 開啟後再依照以下操作流程

放在桌面的路徑
cd C:\Users\user\Desktop\Web_Model_Prediction

安裝依賴:
pip install -r requirements.txt

啟動 Flask 應用程式:
python app.py

打開瀏覽器訪問：http://127.0.0.1:5000/
服務將預設在 https://www.google.com/search?q=http://127.0.0.1:5000/ 運行。
