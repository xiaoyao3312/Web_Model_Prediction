背景顏色調整小工具(Color_BG_Control) 使用方法（任意頁面）
引入 CSS：
    <link rel="stylesheet" href="{{ url_for('static', filename='css/Color_BG_Control.css') }}">
引入 JS（放在 <body> 結尾）：
    <script src="{{ url_for('static', filename='js/Color_BG_Control.js') }}"></script>

專案操作手冊 - Web 模型預測平台
放在桌面的路徑
cd C:\Users\user\Desktop\Web_Model_Prediction

安裝依賴:
pip install -r requirements.txt

啟動 Flask 應用程式:
python app.py

打開瀏覽器訪問：http://127.0.0.1:5000/
服務將預設在 https://www.google.com/search?q=http://127.0.0.1:5000/ 運行。

如果模型首次放入 需要訓練拿到權重
python churn_bank_train.py --train_file "C:\Users\user\Desktop\Web_Model_Prediction\data\datasets\train.csv" --test_file "C:\Users\user\Desktop\Web_Model_Prediction\data\datasets\test.csv"
