🚀 CSS 核心能力總結 (Core Capabilities Summary)
I. 視覺與外觀 (Visual Aesthetics)
    🎨 顏色與背景
        color: 設定文字、圖標的前景色。
        background-color: 設定元素的背景色。
        background-image: 使用圖片作為背景。
        background-repeat, background-position, background-size: 控制背景圖片的重複、位置和大小。
        mix-blend-mode: 設定元素內容如何與背景混合 (如 Photoshop 模式)。

    🖼️ 邊框、圓角與陰影
        border: 設定邊框的寬度、樣式和顏色。
        border-radius: 設定元素的圓角程度。
        outline: 元素外層的輪廓線 (不佔空間)。
        box-shadow: 為元素添加陰影，增加立體感。
        text-shadow: 為文字添加陰影。

    👁️ 可見性與交互
        opacity: 設定元素的整體透明度 (0.0 到 1.0)。
        visibility: hidden: 隱藏元素，但仍佔據空間。
        display: none: 隱藏元素且不佔據空間。
        cursor: 改變滑鼠懸停在元素上時的游標圖案。

II. 文字與排版 (Typography)
    ✍️ 字體與樣式
        font-family: 設定字體（如 Arial, serif, 'Noto Sans TC'）。
        font-size: 設定文字的大小。
        font-weight: 設定文字的粗細（如 bold, 700）。
        font-style: 設定文字是否為斜體 (italic)。
        text-transform: 將文字轉換為大寫、小寫或首字母大寫。

    📏 間距與對齊
        line-height: 設定文字的行高。
        letter-spacing: 設定字母之間的間距。
        word-spacing: 設定單詞之間的間距。
        text-align: 設定文字在容器中的水平對齊方式。
        vertical-align: 設定元素或文字的垂直對齊方式。

    📜 溢出與換行處理
        white-space: 控制文字溢出時是否換行。
        overflow-wrap / word-break: 控制長單詞如何斷開。
        text-overflow: 設定文字溢出時是否顯示省略號 (...)。

III. 佈局與定位 (Layout & Positioning)
    📐 尺寸與模型
        display: 定義元素的佈局模型 (block, inline, flex, grid, none)。
        width, height: 設定元素的寬度和高度。
        min-width, max-width, min-height, max-height: 設定元素的最小/最大尺寸。

    📦 空間與溢出
        margin: 設定元素邊框以外的外邊距。
        padding: 設定元素內容與邊框之間的內邊距。
        overflow: 控制內容溢出元素框時的行為 (hidden, scroll, auto)。

    📍 定位與層級
        position: 設定元素的定位類型 (static, relative, absolute, fixed, sticky)。
        top, bottom, left, right: 配合 position 屬性精確定位元素。
        z-index: 設定定位元素在垂直 (Z 軸) 堆疊時的順序。

IV. Flexbox (一維佈局)
    🔲 容器屬性
        display: flex: 將元素設為彈性容器。
        flex-direction: 定義主軸方向 (row, column)。
        justify-content: 主軸上的對齊方式。
        align-items: 交叉軸上的對齊方式。
        flex-wrap: 控制 Flex 項目是否換行。

    🔸 項目屬性
        flex-grow / flex-shrink / flex-basis: 控制擴張、收縮和初始大小。
        order: 改變 Flex 項目在視覺上的順序。
        align-self: 覆寫單個 Flex 項目在交叉軸上的對齊方式。

V. Grid (二維佈局)
    🌐 容器屬性
        display: grid: 將元素設為網格容器。
        grid-template-columns / grid-template-rows: 定義網格的列數和行數。
        grid-gap / gap: 定義網格線之間的間距。

    █ 項目屬性
        grid-column / grid-row: 定義網格項目所佔據的列數和行數範圍。
        grid-area: 為網格項目命名，用於簡化佈局。
        place-items / place-content: 在網格容器內對齊項目。

VI. 響應式與動畫 (Responsiveness & Animation)
    📱 響應式控制
        @media 規則 (max-width, min-width): 根據不同的設備或視窗尺寸應用不同的樣式。
        視口單位 (vh, vw, vmin, vmax): 基於瀏覽器視口高度或寬度的相對單位。

    🔄 動畫與過渡
        transform: 讓元素進行 2D 或 3D 轉換 (旋轉、縮放、平移等)。
        transition: 設定屬性變化時的過渡動畫效果（如速度、時間）。
        @keyframes 規則: 定義複雜的、多步驟的 CSS 動畫序列。
        animation: 應用和控制 @keyframes 定義的動畫。
